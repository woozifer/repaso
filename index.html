Comandos Básicos de Git

Comando
Descripción
git init
Inicializa un repositorio local de Git en la carpeta actual.
git clone [url]
Clona un repositorio remoto a tu máquina local.
git add .
Agrega todos los archivos modificados al área de preparación.
git commit -m "mensaje"
Registra los cambios en el historial de Git con un mensaje descriptivo.
git push origin main
Sube los cambios al repositorio remoto en la rama main.
git pull origin main
Descarga y fusiona los cambios del repositorio remoto en la rama actual.
git status
Muestra el estado actual del repositorio (archivos modificados, en staging, etc.).
git log
Muestra el historial de commits.
git remote
Lista las conexiones remotas del repositorio (como origin).
git remote add origin [url]
Vincula tu repositorio local con uno remoto en GitHub.
